# Announcement draft

> **Labcoat is now Rust-first** 🥼
>
> The native `labcoat` CLI now owns the complete Alkanes contract workflow:
> project scaffolding, Rust contract tests, compilation, local Bitcoin
> regtest orchestration, wallets, deployment, calls, simulation, and traces.
>
> Highlights:
>
> - `labcoat init` creates a complete Rust contract project.
> - `labcoat test` executes WASIp1 contracts through `labcoat-test`.
> - `labcoat up` starts the full local Alkanes devnet.
> - `labcoat.lock` records deployments by network.
> - Every command supports stable JSON envelopes.
> - `labcoat mcp serve` exposes the workflow over MCP.
> - The former TypeScript SDK and scaffolder are retired.
>
> Install and migrate: <https://github.com/jonatns/labcoat#readme>
